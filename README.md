# FF14 Cosmo Splendor Points Calculator (コスモ技巧点計算機)

FF14のコスモの技巧点（50万点×8ジョブ = 合計400万点）を効率的に達成するための、周回数および所要時間計算ツールです。

## 主な機能

- **8ジョブの進行管理**: クラフター8ジョブそれぞれの進捗状況を個別にトラッキング。
- **リアルタイム周回・時間計算**: 現在点、1回あたりの点数、1回あたりの作業時間を入力するだけで、残り周回数と残り時間を自動算出。
- **目標シミュレーター**: 1日の予定プレイ時間を設定すると、完成までの目安日数をリアルタイムで予測。
- **一括適用**: マクロや製作ルートが同じ場合、すべてのジョブに点数・分数をワンクリックで一括適用。
- **自動保存 (localStorage)**: ブラウザを閉じても入力データが保持されます。

## 使用方法

`index.html` ファイルをブラウザで開くだけで動作します。サーバー構築などは不要です。
